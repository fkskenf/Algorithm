# 행렬의 회전
- 공식 : (x, y) -> (y, N-i-x)
1. N : 배경 사각형의 한변의 길이 
> 위 사진이 기준일때 N =3 이다.
2. i : 90도씩 회전 반복 수
> i = 1이면 90도 회전 <br>
> i = 2이면 180도 회전 <br>
> i = 3이면 270도 회전 <br>
> i = 4이면 360도 회전 <br>
