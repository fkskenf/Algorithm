# 행렬의 회전
- 공식 (우측회전)
> 1. 90도 회전  : (x, y) -> (y, N-1-x)
> 2. 180도 회전 : (x, y) -> (N-1-x, N-1-y)
> 3. 270도 회전 : (x, y) -> (N-1-y, x)
- 공식 (좌측회전)
> 1. 90도 회전  : (x, y) -> (N-1-y, x)
> 2. 180도 회전 : (x, y) -> (N-1-x, N-1-y)
> 3. 270도 회전 : (x, y) -> (y, N-1-x)

1. N : 배경 사각형의 한변의 길이 
> 위 사진이 기준일때 N =3 이다.

2. 팁: 해당 공식을 직접 세울때 : 가운데 값을 기준으로 세우는 것이 좋다.
